Privacy Policy

Data Collection and Usage

Collected data is used for the sole purpose of one-time processing. Message content is read and processed automatically by a computer.

Purpose of Processing

The purpose of this processing is to scan messages for keywords that a Server Owner has prohibited or added auto-response mechanisms for.

Data Retention

Data is not saved or recorded anywhere. All processing occurs in memory and is discarded immediately after processing completes.

Information We Access

  • Message content (for keyword scanning)
  • Server configuration (for moderation settings)
  • User IDs (for moderation actions)

What We Don't Store

  • Message content after processing
  • User personal information
  • Any logs of user activity

Changes to This Policy

We may update our Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page.
